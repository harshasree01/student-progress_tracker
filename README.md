# student-progress_tracker
the student progress tracker system is a python based mini project designed to store manage,and analyze student academic performane. this system helps to track student details such as name,roll number,subject marks,total marks,percentage,grade,and overall progress in a structured and user-friendly way.

the project is mainly developed to strengthen basic python programming concepts like variables,loops,dictionaries,and functions.it is suitable for beginners and academic mini project submisiions 


FEATURES:
* add student details(Name,rollnumber)
* enter subject-wise marks
* caluclate total marks and percentage
* assign grades based on performance
* display student progress report
* simple and easy-to-undertand console interface
